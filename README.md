Arduino-based robot navigation using command-based motor control.

## Objective

Build a robot that can navigate a predefined path accurately using **command-based motor control**.

The robot must:
- Start from the **Start** point.
- Follow the exact path shown.
- Reach the **Finish** point without deviation.

This challenge tests your understanding of:
- Motor direction control
- Logical sequencing of movements
- Precision in distance-based motion
- Translating real-world dimensions into robot actions

## Problem Statement

You are given a fixed path consisting of straight movements and turns with defined distances (in feet).

Your robot must follow this path step-by-step using **programmatic control only**.

## Path Description

The path consists of:
- Horizontal and vertical movements
- Multiple left and right turns
- Fixed segment lengths: **3 ft, 4 ft, 5 ft, and 6 ft**

The robot must:
1. Start at the **Start** position.
2. Follow the path in the correct order.
3. Reach the **Finish** position.

## System Type

- Open-loop control system
- No sensor feedback
- Movement based purely on timing and logic
- Accuracy depends on calibration and consistency
